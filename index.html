Features

Everything that changes daily habits

Screen Time Limits

Set a daily limit per child, watch live usage progress, and get notified as they approach it.

Schedule a recurring bedtime lock, or start a one-off timer for homework or dinner.

App Blocking

Choose exactly which apps your child can use — enforced in real time, nothing to uninstall.

Content Filtering
